# GCFIS — Product Vision & Roadmap

## Product Vision (2026–2028)

GCFIS is the **plug-and-play AI Sales & Support Agent** that replaces broken website experiences (static pages, forms, basic chatbots) with a single intelligent conversational layer.

It answers instantly, detects intent, remembers context across visits, handles objections, qualifies leads, takes real actions (book meetings, check orders, refresh knowledge), and works 24/7 in English + Hindi/Hinglish.

### Core Promise

> Turn any website (especially WordPress/WooCommerce) into a proactive teammate that feels like your best salesperson — cheaper than humans, smarter than static tools, and always available.

## Roadmap

| Phase | Timeline | Key Capabilities | Business Impact |
|-------|----------|-----------------|-----------------|
| **MVP (Current)** | Now | Basic widget + backend, Knowledge Base + Firecrawl refresh, Ollama/local LLM, persistent memory, simple tools | Instant answers, 70–80% query deflection, easy WordPress install |
| **v1 — Sales Agent** | 1–3 months | Advanced intent detection, objection handling, lead scoring, multi-tool orchestration (Calendly, Razorpay, inventory), human handoff with context | 3–5× higher conversion, qualified leads only |
| **v2 — Multi-Channel & Learning** | 3–6 months | WhatsApp + Voice support, auto KB improvement from interactions, A/B testing of responses, analytics dashboard | 60–70% support automation, continuous self-improvement |
| **v3 — Enterprise & Ecosystem** | 6–12 months | Multi-agent system (specialized sub-agents), custom fine-tuning, team collaboration features, API marketplace for skills | Full replacement of support + sales teams for mid-market, white-label options |
| **Long-term** | 12+ months | Autonomous agent loops, cross-business knowledge sharing (with permission), predictive personalization, voice-first for India | Become the default "customer OS" for SMBs globally |
